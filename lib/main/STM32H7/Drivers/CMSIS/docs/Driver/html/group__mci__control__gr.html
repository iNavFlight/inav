Description

Configure and control the MCI using the ARM_MCI_Control.

Many parameters of the MCI driver are configured using the ARM_MCI_Control function.

The various MCI control codes define:

  • MCI Controls configures and controls the MCI interface
  • MCI Bus Speed Mode specifies the bus speed mode
  • MCI Bus Data Width specifies the data bus width
  • MCI CMD Line Mode specifies the CMD line mode
  • MCI Driver Strength specifies the driver strength

Refer to the function ARM_MCI_Control for further details.
