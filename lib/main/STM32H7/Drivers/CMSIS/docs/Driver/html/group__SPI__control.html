Description

Many parameters of the SPI driver are configured using the ARM_SPI_Control function.

The various SPI control codes define:

  • SPI Mode Controls specifies SPI mode
  • SPI Frame Format defines the frame format
  • SPI Data Bits defines the number of data bits
  • SPI Bit Order defines the bit order
  • SPI Slave Select Mode specifies slave select mode
  • SPI Miscellaneous Controls specifies additional miscellaneous controls

Refer to the ARM_SPI_Control function for further details.
