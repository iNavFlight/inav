# FLIGHT CONTROLLER MATEK F405-WING

* Designed specifically for fixed wing INAV setups by MATEK
* http://www.mateksys.com/?portfolio=f405-wing

## HW info

* STM32F405
* MPU6000
* OSD
* BMP280
* 6x UART
* 1x Softserial
* 2x I2C
* 2x Motors & 7x Servos
* 4x BEC + current sensor

### MATEKF405SE_PINIO
Replaces UART 6 Tx with USER 1 for PINIO
